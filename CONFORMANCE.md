# UACP Conformance Registry

Implementations that have passed the UACP conformance test harness.

## How to submit

1. Clone the `fusionlayer/uacp` repo (public — coming soon)
2. Implement the `UACPImpl` interface in `packages/shared/src/uacp/conformance/harness.js`
3. Run `node harness.js L3` (or L2/L1) against your implementation
4. Open a PR adding your entry to the table below
